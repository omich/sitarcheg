## How to build ##


### With CLI ###


1. Go to project root
2. $ mkdir build
3. cmake ../src
4. make


### With GUI ###


1. $ cmake-gui
2. Select source dir: sitar/src
3. Create and select build dir: sitar/build
4. Press 'configure' and select UNIX Makefiles
5. Press 'configure' one more time
6. Press generate
7. Go to buikd dir
9. $ make
